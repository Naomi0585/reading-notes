# Revisions and the Cloud

**What is Version Control?**

<ins> Version Contol </ins> is a system that allows you to revisit various versions of a file or set of files by recording its changes. One of the the most utilized version control system worldwide is **Git**.

## Learning Git ##

One of the commands you can use on git is `cloning`. This command will clone files and all it's versions for a project. The `cloning` command will create a directory called *test* which contains an individualized Git directory. Something convenient about this command is that it automatically checks and retrieves the newest version of the project!

### Some Git Commands ###

* **Snapshot** = `git commit -a`.
* **Pushing Changes** - `git push origin master`.
